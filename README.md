# Ollama_RAG_Restaurant_Review
A local based LLM that can generate restaurant reviews based on a csv dataset

## Preconditions
1. **Ollama** installed and installing any **LLM model** and **embedding model**
2. Install **Pandas**, **langchain_core**, **langchain_ollama** and **langchain_chroma**

## Results
Change the **names** of **LLM model** and **embedding model** to **yours**
Run the project on any Python IDE
